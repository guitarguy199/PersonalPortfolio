ZapChatFirebase Messenger App

ZapChat is an instant messaging app with Firebase integration. This project utilizes Firebase for authentication and database queries. The app checks for authentication on every step of the messaging process, increasing security and allowing for error handling. The chat view updates in real time as new messages come in. ZapChat also features a keyboard manager which keeps the textfield and send button in view even when the keyboard is fully displayed.

This was a fun project, and opened the door for more advanced databasing through Firebase, iCloud, CoreData, and Realm in other apps. Using the MVC design pattern kept everything clean and efficient (especially with a heavyweight package like Firebase), and has proven an important habit to keep.
