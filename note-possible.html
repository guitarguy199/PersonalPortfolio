Note PossibleCoreData and Realm

Note Possible is a checklist and note-taking app modeled off the popular app Clear. Simply create a category, tap in, and start creating notes. Swipe the cells away to clear. The UI features beautiful gradients matching the primary colors of the category cells.

This was great practice with databasing, as the app was created through two methods: first with a CoreData backend and then migrated to Realm.

The databases store information about the category and its child, the list items. It also stores the color properties of the cells, allowing for the child to inherit the color of the parent.
